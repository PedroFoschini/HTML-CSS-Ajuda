Introdução

HTML - HyperText Markup Language ou Linguagem de Marcação de HiperTexto na tradução para o português - é uma ferramenta de marcação de texto para criação de páginas web criada em 1991 por Tim Berners-Lee na Suiça e é escrita em formato de texto livre e sem marcação, ou seja, não há nada além de um texto cru em sua composição. A priori com a inteção de apenas fazer ligações de documentos locais, teve um crescimento exponencial e, apenas um ano após sua criação já carregava a responsabilidade de interligar o mundo tudo, 'WWW', World Wide Web.

Por mais que o HTML seja potente o bastante para ter em seu escopo o conteúdo da página e poder estilizá-la, é muito utilizado e indicado o uso de mais duas ferramentas para facilitar a criação e o escalonamento web, o CSS e o JavaScript. Portanto o HTML fica responsável carregar o conteúdo cru da página, o CSS pela estilização e o JavaScript pela interação e lógica por trás.

Começando o estudo das tags, a estrutura básica de praticamente todas consiste em '<x>' para abrir e '</x>' para fechar, sendo 'x' é uma tag representativa geral, possuindo atributos globais e locais.
O corpo principal da página pode ser dividido em 4 contêiners básicos:

  1. <!DOCTYPE html> : Responsável por mostrar que é uma página web;
  2. <html> : Começo do corpo html;
  3. <head> : Compõe o cabeçalho da página, normalmente informações que não aparecem;
  4. <body> : Todo o corpo da página, ou seja, as informações que irão compor todo o site.

OBS: De todas as tags, somente a (2) será fechada ao fim de todas as outras, ou seja, a última tag é </html>

Tags

Além de praticamente impossível decorar todas as tags, além de não ser necessários, pois existem diversos sites para trazer uma biblioteca com todas as tags, dicas e estrutura html, tal como a página da W3C. Esses tipos de site são de suma importância para o andamento do criador de site, então é sempre bom ter na bagagem alguns dos principais para ajudar quando for necessário.

Como dito acima, a página pode ser subdividida em 4 partes principais e, assim, encorporpar cada subdivisão com as demais tags. Para título de estudo, será apresentado abaixo as tags mais usadas em cada uma dessas partes e, caso haja necessidade ou curiosidade para saber mais é indicado buscar sites (como o W3C) para buscar as outras tags.

  • <!DOCTYPE html> é a tag responsável para indicar ao navegador qual a versão do HTML deve ser usada, por isso é indicado sempre colocá-la na primeira linha de todo documento HTML, e, assim, não há outras tags dentro da sua composição e nem necessidade de fechá-la.

  • <html> é responsável basicamente por tudo. É nela que se encontra a raiz, o motivo, do documento escrito ser um HTML, essencial para qualquer projeto de criação de páginas, sem ela, não há HTML. É dentro dessa tag que se coloca o termo 'lang=x', onde lang representa a linguá em que o site está sendo feito e 'x' deve ser substituido pelo idioma, por exemplo 'pt-br' para português ou 'en' para inglês.
    Diferente das outras, essa tag necessariamente deve ser fechada ao final do 'script' todo, ou seja, é a ultima tag de todo o documento. Assim sua forma é:

    <html lang='pt-br'>

  • <head> é o contêiner responsável por trazer à página os dados estruturais, ou seja, nela não se encontra nada visível, mas sim infraestruturas da página, tal como links e conecções com o CSS e o JavaScript, e o título da página. As tags possíveis no head são:
    title, título que aparece na aba; style, basicamente o que faz o CSS; base , especifica o URL base; link, link's que irão juntar links externos; meta, informações sobre os dados; script, instruções do script pelo JavaScript; noscript, define um conteúdo alternativo a ser exibido para usuários que desabilitaram scripts em seus navegadores ou possuem um navegador que não oferece suporte a scripts.
    A tag será fechada assim que terminar todas as expecificações do head e for começar o body.

  • <body> é o responsável por todo corpo da página, onde serão escritos e estruturados todos os ítens que irão aparecer na página, basicamente tudo que é visível é escrito aqui, lembrando que o HTML, por padrão de simplicidade, contêm apenas o conteúdo da página e poucas estilizações.
    Falar de todas as tags que podem compor o body seria muito extenso, então será descrito os principais e mais usados aqui.
    H1,...,H6 definem os títulos mais importantes da página, sendo do menor para o maior número em grau de descendência de importância; p conterá o parágrafo dos textos que irão ser apresentados; ol/ul são tags que carregam a listagem das páginas, respectivamente listagem ordenada e listagem com 'bolinhas' antes, elas apenas abrem a lista, devem ser acompanhadas por li; e div é usado para criar caixas/contêiners dentro do body, muito usado para escalonar eas páginas.
    Essa tag será fechada quando todos os conteúdos da páginas já tiverem sido descritos e estruturados.
